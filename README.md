# API-Project: FoodMatch

---

## Overview

Swipe and find food spots that match what you're craving! 

## Features

1. User login and sign up with Google or email 
2. Let users swipe on food spots from the Yelp Fusion API fitting their criteria (e.g. location, cuisine, availability for delivery)
3. Store users liked and favorite food spots 
4. Show travel options or Uber/Doordash options for saved places
